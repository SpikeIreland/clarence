1. Overview

The CLARENCE video library provides tutorial videos that help users learn the platform. Videos are managed through the Admin Panel and automatically appear throughout the application when published.

Where videos appear:
•	Training Lobby – Featured videos in "Learn Before You Practice" section
•	Tutorials Tab – Full video library organized by category
•	Help Icons – Small video icons next to features throughout the app


2. How to Upload Videos

Videos are managed via the Admin Panel. Follow these steps to add a YouTube video:
1.	Access the Admin Panel Navigate to clarencelegal.ai/admin/beta-testing and click the Videos tab.
2.	Find the Video Use the category filter or search bar to locate the video placeholder you want to update.
3.	Get the YouTube Video ID From the YouTube URL, copy only the video ID (the part after v=). Example: youtube.com/watch?v=dQw4w9WgXcQ → Copy: dQw4w9WgXcQ
4.	Add the YouTube ID Click the "+ Add YouTube ID" button next to the video, paste the ID, and click Save.
5.	Verify The video status will change to "Published" and the thumbnail will appear. The video is now live throughout the application.

5. Recording Guidelines

To ensure consistent quality across all tutorial videos:
•	Duration: Keep videos between 60-90 seconds unless noted otherwise. Users prefer short, focused content.
•	Screen Recording: Use the actual CLARENCE interface with sample data.
•	Audio: Clear voiceover explaining each step. Avoid background music.
•	Branding: Include CLARENCE logo in corner. Use amber colour accents for training-specific videos. Use slate/blue accents for risk mitigation videos.
•	Terminology: Use "Contract Studio" (not "Negotiation Studio"). Use "customer" and "provider" for party roles. Use "playbook" for risk mitigation rules.
•	Upload: Upload to the CLARENCE YouTube channel, then copy the video ID to the admin panel.
•	Featured Videos: After uploading, mark key videos as "Featured" in the admin panel to highlight them on the Training Lobby.
