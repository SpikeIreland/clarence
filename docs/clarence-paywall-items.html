4. Implementation Notes

Area Detail
Stripe Setup 7 products in Stripe Product Catalogue. Platform Base as a recurring subscription; add-ons as separate recurring products linked to the same customer; Additional Contract as metered/usage-based billing.
Database Entitlement table per organisation: tier, active add-ons, seat count, contract usage, AI interaction count (resettable monthly). Synced via Stripe webhooks.
Paywall Enforcement Middleware or API-level checks against entitlement table. Boolean gates for feature access; counter checks for quotas (contracts, seats, AI interactions).
Upgrade Flow In-app prompts redirect to Stripe Checkout or Stripe Customer Portal for subscription management.
